# cat_vasya_game

## Описание проекта

проект, реализующий базовую механику ролевой игры на Python. Проект демонстрирует принципы объектно-ориентированного программирования: абстрактные классы, наследование, полиморфизм и инкапсуляцию.
В игре реализованы персонажи трёх классов (воин, маг, охотник) с различными характеристиками, а также система заклинаний и маны.

## Игровая механика

- Персонажи имеют 6 базовых характеристик: сила, ловкость, телосложение, мудрость, интеллект, харизма
- Три класса персонажей: воин (warrior), маг (mage), охотник (hunter)
- Система заклинаний: персонажи могут изучать и применять заклинания, тратя ману
- Расчёт характеристик: здоровье, урон и защита рассчитываются по формулам, зависящим от класса

## Архитектура проекта

Проект состоит из трёх модулей, реализованных в одном файле.

### Модуль 1: Абстрактный класс Unit

Абстрактный базовый класс для всех игровых юнитов:

- Хранит 6 базовых характеристик
- Содержит абстрактные методы:
  - calculate_max_health() - расчёт максимального здоровья
  - calculate_damage() - расчёт урона
  - calculate_defense() - расчёт защиты
- Предоставляет методы для работы с заклинаниями:
  - add_spell() - добавить заклинание
  - cast_spell() - применить заклинание

### Модуль 2: Класс Character

Конкретная реализация персонажа, наследующая Unit:

- Поддерживает три класса: warrior, mage, hunter
- Реализует все абстрактные методы с ветвлением по классу персонажа
- Проверяет корректность переданного класса
- Автоматически рассчитывает и сохраняет:
  - Максимальное здоровье
  - Базовый урон
  - Защиту
  - Максимальную ману

Формулы расчёта:

Класс Воин:
- Здоровье: телосложение * 10 + сила / 2
- Урон: сила * 2.2 + телосложение / 3
- Защита: телосложение * 1.8 + сила / 4
- Мана: интеллект + сила / 2

Класс Маг:
- Здоровье: телосложение * 10 + сила / 2
- Урон: интеллект * 2.5 + мудрость / 2
- Защита: мудрость * 1.3 + интеллект / 6
- Мана: интеллект * 3 + мудрость

Класс Охотник:
- Здоровье: телосложение * 10 + сила / 2
- Урон: ловкость * 1.9 + сила / 3
- Защита: ловкость * 1.6 + телосложение / 5
- Мана: ловкость * 1.5 + мудрость / 2

Все результаты округляются вниз до целого числа.

### Модуль 3: Система заклинаний

Абстрактный класс Spell - шаблон для всех заклинаний:
- Хранит: название, урон, стоимость маны
- Абстрактный метод cast() - применение заклинания

Конкретные заклинания:
- Fireball - Огненный шар (урон 35, стоимость 15 маны)
- IceLance - Ледяное копьё (урон 25, стоимость 10 маны)
- LightningBolt - Разряд молнии (урон 40, стоимость 20 маны)
